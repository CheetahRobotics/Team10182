## Installation

This is very easy.

1. Download the Python 2.7 version from here: https://www.continuum.io/downloads
2. Then type:
    `jupyter notebook`
    
3. This will open a browser and put you at the start page for Jupyter. Click on 'New' and then 'Python 2'.
